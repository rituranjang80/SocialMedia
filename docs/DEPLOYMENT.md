# Deployment Guide

Run BrightBean Studio with **one command** (UI + Agent API + Swagger + background workers).

## One-click: Docker (recommended)

Starts: **Web UI**, **Agent API**, **Swagger**, **Postgres**, **Tailwind watcher**, and **parallel background workers**.

### Windows

```powershell
.\scripts\start.ps1
```

### Linux / macOS

```bash
./scripts/start.sh
# or
make run
```

### URLs after start

| Service | URL |
|---------|-----|
| Web UI | http://localhost:8000 |
| Swagger | http://localhost:8000/api/v1/docs |
| OpenAPI JSON | http://localhost:8000/api/v1/openapi.json |
| Agent API | http://localhost:8000/api/v1/ |
| Health | http://localhost:8000/health/ |

Create admin:

```bash
docker compose exec app python manage.py createsuperuser
```

Stop:

```bash
docker compose down
```

## One-click: Kubernetes

### Windows

```powershell
.\scripts\start-k8s.ps1
```

### Linux / macOS

```bash
./scripts/start-k8s.sh
# or
make run-k8s
```

Port-forward:

```bash
kubectl -n brightbean-studio port-forward svc/brightbean-app 8000:80
```

Manifests: `deploy/kubernetes/`

| Resource | Purpose |
|----------|---------|
| `postgres` StatefulSet | Database |
| `brightbean-app` Deployment | Gunicorn (UI + API + Swagger) |
| `brightbean-worker` Deployment | Background jobs (default 2 replicas) |
| `brightbean-migrate` Job | Runs migrations on deploy |
| Ingress | Routes `/`, `/api`, `/oauth`, `/.well-known` |

## Manual Docker Compose

```bash
cp .env.example .env
# Set DATABASE_URL=postgres://postgres:postgres@postgres:5432/brightbean
docker compose up -d --build --scale worker=2
```

Dev override (`docker-compose.override.yml`) adds:
- `runserver` instead of Gunicorn
- Live code volume mounts
- Tailwind CSS watcher

Production:

```bash
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build
```

Adds Caddy (HTTPS) and Gunicorn with configurable workers.

## Configuration (.env)

### Core

| Variable | Description |
|----------|-------------|
| `SECRET_KEY` | Django secret (auto-generated by start scripts) |
| `ENCRYPTION_KEY_SALT` | Token/credential encryption salt |
| `DATABASE_URL` | PostgreSQL connection string |
| `APP_URL` | Public URL (OAuth redirects, emails) |
| `ALLOWED_HOSTS` | Comma-separated hostnames |

### Multitenant & parallelism

| Variable | Default | Description |
|----------|---------|-------------|
| `MULTITENANT_ENABLED` | `true` | Informational; org/workspace isolation always active |
| `WORKER_REPLICAS` | `2` | Background worker count (Docker `--scale worker=N`) |
| `PUBLISHER_MAX_CONCURRENT_PUBLISHES` | `10` | Parallel platform publishes per poll |
| `PUBLISHER_MAX_CONCURRENT_POSTS` | `4` | Parallel post groups in flight |
| `GUNICORN_WORKERS` | `4` | HTTP worker processes (production) |
| `GUNICORN_THREADS` | `2` | Threads per Gunicorn worker |

### API / Swagger

| Variable | Default | Description |
|----------|---------|-------------|
| `AGENT_API_DOCS_ENABLED` | `true` | Show Swagger UI at `/api/v1/docs` |
| `API_CONFIG_PATH` | `config/api_defaults.json` | JSON file for Swagger dropdowns & defaults |
| `API_INBOX_DEFAULT_LIMIT` | from JSON | Override inbox page size |
| `API_INBOX_MAX_LIMIT` | from JSON | Max inbox page size |
| `API_ANALYTICS_DEFAULT_DAYS` | from JSON | Default analytics window |

Edit `config/api_defaults.json` (or mount your own file at `API_CONFIG_PATH`), then `docker compose restart app worker` вҖ” **no image rebuild**.

Example custom config mount in `.env`:

```env
API_CONFIG_PATH=./my-api-config.json
```

Docker Compose bind-mounts this file to `/app/config/api_runtime.json` inside containers.

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Posts not publishing | Ensure worker is running: `docker compose logs worker` |
| Swagger 404 | Set `AGENT_API_DOCS_ENABLED=true`, restart app |
| Tailwind not compiling | `docker compose logs tailwind` (dev compose only) |
| Migrations failed | `docker compose run --rm migrate` |
